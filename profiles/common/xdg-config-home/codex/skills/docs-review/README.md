```text
$docs-review docs/spec과 docs/architecture를 검토하고 직접 수정해줘.
```

다른 경로의 프로젝트라면 루트를 명시합니다.

```text
$docs-review /path/to/project의 docs/spec과 docs/architecture를 검토하고 수정해줘.
```

수정 없이 리뷰만 받을 수도 있습니다.

```text
$docs-review docs/spec과 docs/architecture를 수정하지 말고 문제만 보고해줘.
```
